# Sejarah Persib Bandung

Situs arsip & sejarah **Persib Bandung** yang dibuat oleh bobotoh.

Proyek ini dibuat untuk dua tujuan pribadi:

1. **Mengarsipkan sejarah Persib** — mengumpulkan dan merapikan jejak klub dari era ke era: kompetisi, gelar, pemain, pelatih, dan momen-momen bersejarah, dari berbagai sumber terbuka.
2. **Sarana belajar** — bereksperimen dengan Nuxt, Nuxt UI, dan Directus sebagai headless CMS.

Bersifat *fan-made*, **tidak berafiliasi dan tidak mewakili** PT Persib Bandung Bermartabat atau manajemen klub. Data disusun dari sumber terbuka dan terus diperbarui seiring proses verifikasi, jadi mungkin masih ada yang belum akurat.

## Teknologi

- **Frontend** — Nuxt 4 + Nuxt UI v4 (`web/`)
- **CMS** — Directus (dijalankan via Docker), dengan database Supabase Postgres (`cms/`)

## Status

Masih dalam pengembangan aktif dan belum ditujukan untuk kolaborasi atau di-fork. Kalau ada koreksi data sejarah, silakan sampaikan lewat form **Kritik & Saran** di situsnya.
